Task 2: Data Cleaning & Exploratory Data Analysis (EDA)

📌 Objective

The objective of this task is to clean the dataset and perform exploratory data analysis (EDA) to discover patterns and insights.

---

📊 Dataset Used

- Titanic Dataset (CSV file)
- Contains details like passenger age, gender, class, and survival status

---

🛠 Tools & Libraries

- Python
- Pandas
- Matplotlib
- Seaborn

---

🔧 Data Cleaning Steps

- Checked for missing values in the dataset
- Filled missing Age values using mean
- Removed unnecessary column (Cabin)

---

📈 Data Analysis & Visualizations

1. Survival Count

- Shows number of passengers who survived vs not survived

2. Survival vs Gender

- Compares survival based on gender
- Helps identify which gender had higher survival rate

3. Correlation Heatmap

- Displays relationships between numerical features
- Helps understand how variables are related

---

📌 Key Insights

- Females had a higher survival rate than males
- Dataset contained missing values which were handled
- Visualization made it easier to understand patterns
